# Raphaël-ZP

A little plugin for [Raphaël](http://raphaeljs.com/) version 2.1.0 and above to
allow click and drag panning as well as zooming.

## Acknowledgements

Forked from [Raphaël-VEMap](https://github.com/christocracy/raphael-vemap).
